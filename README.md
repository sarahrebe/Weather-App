# 🌤 Weather App

A modern Android weather application built with Kotlin and Android Studio.

## Features
- 🌍 Search weather by city
- 📍 Use current GPS location
- 🌡 Current temperature
- 💧 Humidity
- 🌬 Wind speed
- 👀 Visibility
- 🌅 Sunrise & Sunset
- 🕒 Hourly Forecast
- 📅 5-Day Forecast

## Technologies
- Kotlin
- Android Studio
- XML
- Material Design 3
- OpenWeather API
- Gradle

## Download
Download the latest APK from this repository and install it on any Android device.
